# AI-Enabled Internal Fruit Decay Detection System

**Indian Patent Application No. 202541129907** | Filed: December 2025

An AI-powered non-destructive system that detects internal fruit decay using **Terahertz Reflection Imaging** fused with **Multispectral Imaging** and multimodal deep learning.

---

## Overview

Post-harvest losses due to internal decay are a major challenge in agriculture. This project proposes a novel system that penetrates fruit tissue using Terahertz waves and analyzes surface/sub-surface features using multispectral imaging. These are fused using AI models for real-time healthy vs. decayed classification.

**Role**: Student Team Lead & Co-Inventor

---

## Key Features

- Non-destructive internal decay detection
- Multimodal feature fusion (spatial + spectral + frequency domain)
- Real-time classification and automated sorting capability
- Hardware integration: Terahertz module + Multispectral camera + Embedded processing

---

## System Architecture

- **Terahertz Reflection Module**: Captures internal structural variations
- **Multispectral Imaging Unit**: Extracts biochemical and textural features
- **AI Fusion Model**: Multimodal deep learning network (CNN-based feature extractors + fusion layer)
- **Output**: Healthy / Decayed classification with confidence score
